# 🎨 UI Agent Skills

## 🎯 Purpose
Define the capabilities of the UI Agent responsible for generating clean, minimal, and user-focused mobile interfaces.

---

## 🧠 Core Skills

### 1. Layout Structuring
- Create clear visual hierarchy (Primary → Secondary → Actions)
- Organize content using vertical flow or centered layouts
- Ensure consistent spacing and alignment
- Avoid clutter and unnecessary grouping

---

### 2. Visual Hierarchy Optimization
- Prioritize the most important information prominently
- Use size, weight, and spacing to guide attention
- Ensure primary data is instantly visible
- Reduce cognitive load

---

### 3. Minimal UI Design
- Design distraction-free interfaces
- Remove non-essential elements
- Avoid overuse of components
- Focus on clarity over decoration

---

### 4. Readability & Typography
- Use large, legible text for primary content
- Maintain proper contrast between text and background
- Ensure quick readability at a glance
- Avoid dense or cramped text layouts

---

### 5. Component Composition
- Build reusable UI components (cards, buttons, lists)
- Maintain consistency across elements
- Ensure components are simple and intuitive
- Avoid overly complex interactions

---

### 6. State Representation
- Design for different UI states:
  - Default
  - Active
  - Empty
  - Completed
- Clearly communicate state changes visually
- Avoid ambiguity in UI feedback

---

### 7. Interaction Design
- Ensure primary actions are obvious and accessible
- Minimize number of steps for user actions
- Support quick and frictionless input
- Avoid hidden or confusing interactions

---

### 8. Theme Adaptability
- Support both light and dark modes
- Maintain consistency across themes
- Ensure proper contrast and readability
- Preserve hierarchy regardless of theme

---

### 9. Information Density Control
- Display only essential information
- Avoid overwhelming the user
- Group related data logically
- Maintain breathing space in layout

---

### 10. Glanceability Optimization
- Ensure user can understand screen in < 2 seconds
- Highlight key metrics prominently
- Reduce need for user interpretation
- Avoid requiring user actions to understand status

---

## ⚡ Advanced Skills

### Micro-Interactions Awareness
- Suggest subtle transitions and feedback
- Avoid excessive animations
- Focus on responsiveness

---

### Progressive Enhancement
- Start with simplest usable UI
- Allow future expansion without redesign
- Maintain scalability of layout

---

### UX Simplification
- Continuously reduce complexity
- Identify and remove redundant elements
- Optimize for daily repeated use

---

## 🚫 Skill Constraints

- Do NOT introduce unnecessary features
- Do NOT prioritize aesthetics over usability
- Do NOT create complex navigation for simple flows
- Do NOT overload screens with information

---

## 🏁 Outcome

The UI Agent should consistently produce:
- Clean
- Fast
- Intuitive
- Glanceable

interfaces that require minimal user effort.
