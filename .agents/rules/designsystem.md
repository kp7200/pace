# 🎨 Design System (Minimal — AI Agent Guidelines)

## 1. Core Design Identity
- Use **warm cream background (`#F3F0EE`)** instead of pure white
- Prefer **soft, rounded UI**:
  - 20px → buttons
  - 40px → containers / cards
  - 999px → pills
  - 50% → circles
- Use **circular images** for key visuals
- Attach **floating circular CTA (→)** to image edges when applicable
- Maintain **minimal, airy layouts** with generous whitespace
- Primary CTAs must be **black**, not colorful

---

## 2. Color System
### Primary
- **Ink Black (`#141413`)** → text, primary buttons
- **Canvas Cream (`#F3F0EE`)** → main background

### Accent
- **Signal Orange (`#CF4500`)** → ONLY for consent/legal actions (rare use)

### Supporting
- **Gray tones** → secondary text
- **Link Blue (`#3860BE`)** → links only

### Rules
- ❌ No gradients
- ❌ No bright multi-color UI
- ✅ Depth comes from spacing + shadows

---

## 3. Typography
- Use **single font family** (clean geometric sans)

### Hierarchy
- H1 → large, tight spacing
- H2 → section headings
- H3 → card titles
- Body → medium weight, highly readable

### Rules
- Use **tight letter-spacing for headings**
- Avoid excessive font variety
- Avoid unnecessary uppercase (except small labels)

---

## 4. Components

### Buttons
- **Primary Button**
  - Background: Black
  - Text: Cream
  - Radius: 20px

- **Secondary Button**
  - White background + black border

- **Avoid**
  - Bright colored CTA buttons (except consent)

---

### Cards & Images
- Use **circular or pill shapes**
- Avoid standard rectangles
- Keep **content minimal per card**
- Prefer **image-first layouts**

---

### Navigation
- Use **floating pill-style navbar**
- Keep it **light, minimal, and uncluttered**
- Max 5–6 primary items

---

## 5. Layout Principles
- Use **8px spacing system** (8 / 16 / 24 / 32 / 48 / 64+)
- Keep **large vertical spacing between sections**
- Avoid dense layouts
- Prefer:
  - 1–2 elements per section
  - Strong focus areas

---

## 6. Depth & Elevation
- Use **soft, subtle shadows only**
- Avoid sharp or heavy shadows
- Most elements should feel **flat on canvas**

---

## 7. Do’s ✅
- Use cream background consistently
- Use circular imagery
- Keep UI minimal and focused
- Use black as primary CTA color
- Maintain strong spacing and hierarchy

---

## 8. Don’ts ❌
- Don’t use pure white backgrounds
- Don’t use small border radius (8–12px)
- Don’t add unnecessary colors
- Don’t overcrowd layouts
- Don’t mix multiple fonts
- Don’t use gradients

---

## 9. Responsiveness
- Mobile → single column layout
- Tablet → simple grid (2 columns)
- Desktop → asymmetrical but clean layouts
- Maintain:
  - Large touch targets (≥ 44px)
  - Consistent spacing scale

---

## 🎯 Principle Summary
> Clean. Warm. Rounded. Minimal.  
> Focus on **clarity, spacing, and softness**, not decoration.
